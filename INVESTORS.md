# PAiERA - Investment Overview

## Executive Summary
PAiERA is the first 100% offline personal AI assistant that runs locally on user's computer. Unlike cloud AI (ChatGPT), we process everything locally - no data leaves the device.

## Problem
Cloud AI has fundamental privacy issues:
- User data sent to servers
- Companies train on private conversations  
- Internet required
- No long-term memory

## Solution
Personal AI that:
- Runs locally (Mac/Windows)
- Works without internet
- Remembers user's entire life
- Zero data collection

## Technology
**7-Layer Memory Architecture:**
1. Identity - User profile
2. Facts - Knowledge database
3. Emotions - Mood tracking
4. Timeline - Event history
5. Graph - Entity relationships
6. Insights - Pattern detection
7. Predictions - Future planning

**Tech Stack:**
- Backend: Node.js + Express
- Frontend: React
- AI: Qwen2.5-14B (local)
- Storage: File-based (offline-first)
